# Java 2D

Usando la **API 2D** de Java para mostrar las siguientes ***figuras***:

>  - Triángulo
>  - Rectángulo
>  - Pentágono
>  - Círculo
>  - Texto con relleno

Aplicando a cada figura las siguientes ***transformaciones*** y ***efectos de diseño***:

> - Traslación
> - Escalamiento
> - Rotación
> - Relleno
> - Degradado
> - Relleno con imagen
